Architecture SaaS multi-tenant avec cloisonnement logique par société (tenantId obligatoire sur toutes les entités métier) et super-admin global. Application web responsive (SPA + API REST). Authentification sécurisée (JWT ou OAuth2), gestion RBAC fine (superAdmin, companyAdmin, agent, comptable, client). Base de données relationnelle (ex: PostgreSQL) avec indexation sur disponibilités véhicules et performances reporting. Stockage sécurisé des documents (contrats PDF, états des lieux, photos) via object storage. Génération de PDF serveur pour factures légales avec numérotation séquentielle par société. Intégration prestataire paiement (CB + empreinte bancaire) + gestion statuts transactionnels. Système de tâches planifiées pour pénalités automatiques, rappels et notifications (email/SMS via provider externe). Journal d’audit horodaté pour actions sensibles. Moteur de tarification paramétrable (saisonnalité, options, promotions). Gestion calendrier temps réel avec verrouillage transactionnel pour éviter double réservation. Conformité RGPD (consentement, droit à l’oubli, export données). Architecture prête pour scalabilité horizontale (containers, API stateless). API versionnée prévue pour phase 2. Sauvegardes automatiques, chiffrement des données sensibles au repos et en transit (TLS).
